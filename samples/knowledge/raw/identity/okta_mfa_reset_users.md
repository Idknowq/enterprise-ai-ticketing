# Reset multifactor authentication for users

You can reset multifactor authentication (MFA) for single users. Resetting MFA means that the user's enrollment in their authenticators is canceled and they have to enroll in them again.

## Reset authenticators for a single user

This action resets any configured authenticators that you select for an individual user. The user must set up their authenticators again.

- In the Admin Console , go to Directory People .

- Click the user whose multifactor authentication you want to reset.

- Click More Actions Reset Authenticators .

- Select the authenticators that you want to reset, and then click Reset Selected Authenticators . A confirmation prompt appears.

- Click Reset . This action can't be undone.
